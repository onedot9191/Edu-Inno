# 합리적 선택하기 🛒

초등학생 4학년 사회과 '합리적 선택' 학습을 위한 교육용 웹 앱입니다.

## 🚀 시작하기

### 1. 의존성 설치

```bash
npm install
```

### 2. OpenAI API 키 설정

프로젝트 루트 디렉토리에 `.env` 파일을 생성하고 아래 내용을 입력하세요:

```env
VITE_OPENAI_API_KEY=your_openai_api_key_here
```

> ⚠️ **보안 주의**: `.env` 파일은 절대 Git에 커밋하지 마세요! 이미 `.gitignore`에 추가되어 있습니다.

### 3. 개발 서버 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:5173`으로 접속하세요.

## 📚 기능

### 1️⃣ 물건 입력 & 맞춤형 AI 예산 설정
- 사용자가 원하는 물건을 입력합니다
- **AI가 물건의 종류를 분석하여 적절한 예산을 설정**합니다
  - 예: "과자" → 5,000~15,000원, "게임기" → 30,000~80,000원
- 물건에 어울리는 현실적인 예산 범위 내에서 랜덤 설정
- 실제 상황처럼 정해진 예산 내에서 선택하는 경험 제공

### 2️⃣ AI 인터넷 조사 & 전략적 쇼핑 대안 제시
- **"AI가 인터넷을 뒤지는 중..."** 실감나는 로딩 애니메이션
  - 🛒 쇼핑몰 검색 중...
  - 💰 가격 비교 중...
  - ⭐ 리뷰 분석 중...
- **전략적으로 구성된 3가지 대안** (순서는 무작위로 섞임):
  1. **매력적이지만 예산 초과**: 모든 기준을 만족하지만 돈이 부족한 선택
  2. **예산 내 대안 A**: 특정 기준들은 우수하나 다른 기준들은 아쉬운 선택
  3. **예산 내 대안 B**: 대안 A와 정반대의 장단점을 가진 선택 (무엇을 우선시하느냐에 따라 호불호가 갈림)
- **쇼핑몰 스타일 카드 디자인**: 평점, 배송정보, 상세 특징 포함
- 예산 초과 여부를 시각적으로 표시 (⚠️ 예산 초과 / ✅ 예산 내)
- 학생들이 "어떤 기준을 더 중요하게 볼 것인가"를 고민하게 만드는 구조

### 3️⃣ 맞춤형 평가 기준 선택
- 학생이 직접 중요하게 생각하는 기준 3가지를 선택합니다
- AI가 추천하는 6가지 기준 중 선택하거나 직접 입력 가능
- 추천 기준: 💰 가격, 🎨 디자인, 🌱 환경, ⚡ 성능, 📏 크기, 💪 튼튼함
- "✏️ 추가하고 싶은 기준이 있나요?" - 직접 입력 가능

### 4️⃣ AI 넛지와 함께하는 별점 평가
- 선택한 3가지 기준으로 동적으로 생성되는 평가표
- 각 기준마다 1~5점 별점을 매길 수 있습니다
- **🤖 AI 넛지 기능**: 예산 초과 물건에 가격 만점을 주면 AI가 팝업으로 부드럽게 개입
  - "어? 가진 돈보다 비싼데 가격 점수가 만점이라고? 다시 생각해볼까?"
- 총점이 자동으로 계산되어 최선의 선택을 도와줍니다

### 5️⃣ 최종 확인 & 예산 관리 교육
- **마지막 확인 팝업**: "정말 이 물건이 최고의 선택일까? 총점이 가장 높은지 확인해봐!"
  - 3가지 대안의 총점을 한눈에 비교
  - 한 번 더 생각할 기회 제공
- **⚠️ 예산 초과 경고 시스템**: 예산을 초과하는 대안을 선택한 경우
  - "구매 불가능!" 팝업으로 현실적인 제약 체험
  - 가진 돈, 물건 가격, 부족한 금액을 시각적으로 비교
  - "합리적 선택은 예산 안에서 가장 만족스러운 물건을 고르는 것" 교육
  - 예산 내 다른 대안 선택 유도

### 6️⃣ 학습 정리 & 4학년 수준 AI 피드백
- **학습 정리 화면**: "오늘의 쇼핑을 정리해 볼까요?"
  - 문장 완성하기: "나에게 합리적 선택이란 [  ](이)다."
  - 문장 완성하기: "왜냐하면 [  ]기 때문이다."
- **4학년 교과서 수준 AI 피드백** (상투적 칭찬 금지):
  - 학생이 쓴 핵심 단어를 따옴표로 인용하며 구체적으로 분석
  - 정의(A)와 이유(B)의 논리적 연결성 평가
  - 쇼핑 체험(선택한 물건, 예산, 평가 기준)과 연결 지어 설명
  - **용어 사용 규칙**:
    * ✅ 허용 (적극 권장): '합리적 선택', '절약', '만족감', '선택 기준'
    * ❌ 금지: '기회비용', '효용', '소비 성향' 등 어려운 전문 용어
    * 대체 표현: '기회비용' → "아쉽게 포기한 다른 물건"

### 7️⃣ 초등 4학년 눈높이 맞춤 '절제된 키치' 디자인
- **진행 막대**: 화면 상단에 현재 단계(n/6단계)를 시각적으로 표시
- **AI 캐릭터 가이드**: 매 단계마다 귀여운 말풍선으로 친절하게 안내
- **배경**: 연한 파스텔톤 모눈종이 패턴으로 재미있는 느낌
- **컨텐츠 영역**: 흰색 배경 + 둥근 모서리로 가독성 최대화
- **스티커 스타일**: 중요 정보는 다이어리 스티커처럼 입체감 있게 강조
- **왕방울 버튼**: 크고 통통한 버튼에 누르는 재미를 더하는 애니메이션
- **타이포그래피**: 
  - 본문: Pretendard 고딕체로 읽기 편하게 (기본 18px 이상)
  - 제목: Gamja Flower 손글씨 폰트로 귀엽게
- **컬러 팔레트**:
  - 배경: 연한 보라/하늘색 (#F3E5F5, #E3F2FD)
  - 강조: 네온 핑크, 형광 라임, 진한 노랑 팝 컬러
  - 텍스트: 고대비로 선명하게
